# CI5 sherlock — website

Static d3.js website that surfaces the per-cancer reports plus the all-sites overview for non-technical readers.

## Files

```
site/
  index.html          overview page (README + 10 spotlight cards)
  cancer.html         per-cancer page; URL hash selects code (e.g. cancer.html#code=32)
  data.js             generated by scripts/04_build_site.R — full payload
  css/style.css
  js/charts.js        d3 line-chart helper
  js/app.js           page render + nav
```

## Regenerate `data.js`

After editing `reports/*.md`:

```
Rscript "scripts/04_build_site.R"
```

This re-reads `cache/asr.rds` and the markdown reports and rewrites `site/data.js`.

## Preview

Serve from the `site/` directory (any static server). For example:

```
cd site
python -m http.server 8765
```

then open `http://localhost:8765/index.html`.

The site relies on `data.js` being loaded via a `<script>` tag rather than `fetch()`, so it also opens directly via `file://` if needed.
